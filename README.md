# JavaScript-Game-TicTacToe

> For a complete version of pure HTML/JavaScript, please refer to branch [**`v_html`**](https://github.com/RedCarpG/JavaScript-Game-TicTacToe/tree/v_html)

## Overview

-----------------------------

A **Tic Tac Toe** game designed by `JavaScript` & `React JS` & `TailWind`. Animated mostly by `JavaScript` + `CSS3`.


***This project is for my personal `JS/CSS/HTML/ReactJS/TailWind` learning purpose. All the SVGs and animations are created on my own.***

## 🔮 Overview of the files

-----------------------------

|   |   |
| --------------------------------- |   --------------------------------- |
|   |   |
|  📂 `config` | Configuration files for Plugins |
|  📂 `public` | Folder containning static files |
|  📂 `src` | Folder containning source code for ReactJS |
|  📂 `dist` |  Packed files built by `gulpfiles.js`, containing the filnal files to be used as the web page |
|  📃 `gulpfile.js` |  Gulp config file to build the project into `dist` folder |
|   |   |

For font, I'm using **[KGRedHands](https://www.dafont.com/kg-red-hands.font)**.

Considering that only a small portion of font characters are used, I used `font-spider` to reduce the font file size.
